var, let, const

1. 块级作用域

在ECMAScript 6 标准中,引入了块级作用域的概念  
块级作用域由 {} 包括, if 和 for 语句中的{}同样属于块级作用域

2. let 与 var

  • let 作为es6的语法, 想要JS中引入, 必须使用严格模式
    ‘use strict’

  • var定义的变量, 没有块的概念, 可以跨块访问, 不能跨函数访问
    let定义的变量, 只能在块作用域里访问, 不能跨块访问, 也不能跨函数访问。

3. const

const在定义时必须初始化, 且初始化后不能修改, 不能跨块访问

在声明的常量中保存的仅是对象的指针, 修改对象的属性不会改变对象的指针, 所以是被允许的
